# aws-s3-static-portfolio
AWS S3 Static Portfolio Website 

This repository contains my personal portfolio website developed using HTML, CSS, and JavaScript and hosted using Amazon AWS S3 static website hosting.

Technologies Used
- HTML5
- CSS3
- JavaScript
- Amazon S3 (Static Website Hosting)
- GitHub

Project Structure
- index.html – Main webpage
- style.css – Styling file
- script.js – JavaScript functionality

Project Documentation
A demo video explaining the website setup and hosting process is included.


What I Learned
- AWS S3 bucket creation and configuration
- Static website hosting on AWS
- Managing public access permissions
- Deploying frontend projects to cloud
